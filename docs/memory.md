# Análise de Memória
- Estruturas eficientes são utilizadas ao longo do projeto, visando a uma maior otimização ao longo de todo o programa


# Lições Aprendidas

### O que funcionou bem
A adição de testes que cobrem uma grande quantidade de linhas e branches foi essencial para a identificação de bugs diversos ao longo dos códigos.

### Desafios enfrentados
A utilização das diferentes ferramentas para a medição de tempo decorrido levou a uma curva de aprendizado durante o decorrer do projeto. Além disso, o pensamento para gerar testes que abordem diferetes branches levou tempo e esforço.

### Melhorias futuras
Melhorias futura para o projeto incluem a produção de mais testes, tanto unitários quanto integrados, (sobretudo em casos de borda) para a correção de eventuais bugs encontrados e possíveis erros durante a execução.
